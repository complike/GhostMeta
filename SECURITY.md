Security Policy

Supported Versions:
  Only the latest version of GhostMeta is supported.

Reporting:
  Please DO NOT report security vulnerabilities publicly.
  Use GitHub → Security → Private Vulnerability Reporting.
  Alternatively, contact the author via the email listed in the repository.

Scope:
  GhostMeta is open source (MIT).
  The GhostEngine is proprietary and follows its own internal security processes.

Responsible Disclosure:
  Please allow at least 90 days for analysis and remediation.
  Do not publish vulnerability details without prior coordination.

Anti-Forensics Notice:
  GhostMeta must not be used for illegal activities or to conceal
  criminal behavior. Any misuse is strictly prohibited.
