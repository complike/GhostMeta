Contributing to GhostMeta

Thank you for your interest in contributing to GhostMeta.

Scope of Contributions:
  Contributions are welcome for:
    - bug fixes
    - documentation improvements
    - performance optimizations
    - new JPEG-related forensic features
    - code quality improvements

Not in Scope:
  The GhostEngine is proprietary and not part of this repository.
  Contributions to the GhostEngine, its algorithms, or its internal
  architecture cannot be accepted.

Before Starting Work:
  For major changes or new features, please open an issue first to
  discuss your idea. This avoids duplicated work and ensures that
  the proposed change fits the project direction.

How to Contribute:
  1. Fork the repository
  2. Create a feature branch
  3. Submit a pull request with a clear description

Testing:
  Please test your changes with example JPEG files.
  If available, run:
    - make test
    - or the self-test script provided in the repository

Coding Style:
  - C99
  - clang-format recommended
  - no external dependencies

Security:
  Please do not report vulnerabilities publicly.
  Use GitHub → Security → Private Vulnerability Reporting.

License:
